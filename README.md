# SwirlDesk

SwirlDesk ist ein persönliches Hyprland-Desktop-Setup für Debian mit Fokus auf:

- Hyprland
- Waybar
- Fuzzel
- Dunst
- Kitty
- Wallpaper-Auswahl
- SwirlDesk Control Center
- Theme-System
- Debian/ThinkPad-freundliche Struktur

## Installation

```bash
git clone https://github.com/DEINNAME/swirldesk.git ~/.config/swirldesk
cd ~/.config/swirldesk
./install.sh
